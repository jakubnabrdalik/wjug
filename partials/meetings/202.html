It’s a Docker … It’s a Cloud … It’s a Docker Cloud!

Karol Lotkowski

Termin: 21.03.2017 godz. 18:15

Lokalizacja: MIMUW, sala 3180

O prezentacji:

Po prezentacji będziecie wiedzieć czym jest podejście CaaS i czy Wasze zespoły także powinny zacząć używać Docker Cloud na produkcji. Zdziwicie się, że jest to tak prosta, a zarówno bardzo rozbudowana platforma, która rozwija się w nadzwyczajnym tempie. Zdobędziecie argumenty jak rozmawiać z biznesem, aby przekonać organizację do używania Docker Cloud w Waszych projektach. Przedstawię jak skonfigurować skalowalną usługę w chmurze w 5 minut za pomocą kilku kliknięć.
