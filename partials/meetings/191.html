O prezentacji:

Hibernate was started in 2001 as an alternative to using EJB2-style entity beans. But today EJB is forgotten and we want to make reactive software. How to use functional approach in Object Oriented Mapping? Let’s look at modern Java libraries and know pros and cons of functions, streams and lambdas usage with databases.
