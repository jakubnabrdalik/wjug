Hibernate since 2001. Is time for Cyberpunk 2020?

Tomasz Skowroński

Termin: 22.11.2016 godz. 18:15

Lokalizacja: MIMUW, sala 3180

Sponsor:

O prezentacji:

Hibernate was started in 2001 as an alternative to using EJB2-style entity beans. But today EJB is forgotten and everybody wants to make modern software with Java 8 features. Hibernate is de facto industry standard. But what if it isn’t universal solution for all systems? How to do better declarative or functional querying?

Let’s put aside classic Object-Relational Mapping and look at other, modern capabilities to know pros and cons of DSLs, lambdas and streams usage with relational databases.

Theory, slides, comparisons, but also practical tips and many code examples in IDE.
